# Interactive Image Gallery 🖼️

> **Project Overview**: A fully responsive, interactive Image Gallery featuring a custom Lightbox (modal) view. This project focuses on modern CSS Grid layouts, seamless DOM manipulation, and circular array logic for image navigation.

---

## 🛠️ Tech Stack
* **Markup:** HTML5
* **Styling:** CSS3 (CSS Grid, Media Queries)
* **Scripting:** JavaScript (ES6+)

## 💻 Features
* **Custom Lightbox (Modal):** Users can click on any image to view it in an enlarged, centered modal.
* **Circular Navigation:** Next and Previous buttons allow continuous scrolling through the gallery using modulo arithmetic logic.
* **Responsive CSS Grid:** The gallery dynamically adjusts its column count (from 2 to 4 columns) based on viewport width for an optimal viewing experience on mobile, tablet, and desktop.
* **Event Delegation:** Optimized JavaScript event listeners attached to parent containers for better memory management.

## 🚀 Setup & Installation (Local Development)

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**
   ```
   git clone https://github.com/sahilvish01/Interactive-Image-Gallery.git
   ```
2. **Navigate to the directory:**
   ```
   cd Interactive-Image-Gallery
   ```
3. **Run the application:**
   Open the `index.html` file in your browser, or use the **Live Server** extension in VS Code for real-time updates.
